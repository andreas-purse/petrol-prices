# Session Logs

This folder is automatically updated by a sub-agent after each work session. Each file tracks a feature area or topic:

- Changes made and why
- Patterns discovered
- Failures and how they were resolved
- Decisions and trade-offs

Files are created/updated automatically as work happens. Don't edit these manually.
